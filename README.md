## ⏰ 개발 기간 및 작업 관리

- **1주차** : 다이어그램 작성, 태블릿에서 PC로 전환되는 디자인 전략 기획
- **2주차** : 사용자 인증, API 커스텀 훅, 공통 컴포넌트 제작
- **3주차** : 회원관리, 일정 및 상담 추가 기능 구현
- **4주차** : 일정관리, 회원관리, 센터관리, 마이페이지 기능 구현 및 리팩토링
  <br/>

## 📝 주요 개발 기능

- 관리자 / 직원 로그인 기능
- 회원 / 센터 수강권 / 회원 수강권 관리기능
- 캘린더 및 일정 관리 기능
